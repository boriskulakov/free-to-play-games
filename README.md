## Локальный запуск

Склонируйте репозиторий

```bash
git clone https://github.com/boriskulakov/free-to-play-games
```

Билд и запуск сервера

```bash
npm run build

npm start
```

Откройте [http://localhost:3001](http://localhost:3001) в вашем браузере
